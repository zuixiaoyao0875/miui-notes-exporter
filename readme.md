# MIUI Notes 终极还原引擎 (v31.0 Platinum)

本项目是一套针对小米笔记 (com.miui.notes) 现代 Android 备份协议的底层还原与深度渲染解决方案。它不依赖小米官方恢复环境，直接通过**深度二进制挖掘**与**结构化数据编译**，将私有数据完美迁移为高保真 Markdown 知识库。

## 📖 核心特性与技术架构 (Architecture & Features)

本引擎突破了传统提取工具“只能看字”的局限，彻底解决了文件排序混乱、排版丢失和时间戳乱码的问题。

### 1. 物理切块与深度打捞 (Binary Stream Carving)
针对备份文件（.bak/.tar）中可能存在的非标准压缩或损坏，本引擎采用底层指纹滑窗检索技术：
- `FF D8 FF`: 探测 JPEG 图像起始，打捞至 `FF D9`。
- `89 50 4E 47`: 探测 PNG 图像起始，打捞至 `IEND` 标志位。
- 提取出的媒体资产自动通过 SHA1 哈希值建立全网唯一寻址索引。

### 2. Protobuf 协议反解与智能嗅探
小米笔记以 Protobuf 结构化存储，标签名混乱多变。我们首创了**深度探测算法**：
- **智能时间戳探针**：不再强依赖特定的 `modify_time` 字段，而是使用递归算法全盘扫描 payload，抓取所有 13 位有效时间戳（毫秒级），并取最大值作为**最终更新时间**，彻底消灭 `updated: N/A` 的问题。
- **文件系统友好化**：提取原笔记的 LUID 锚点，并自动补齐为三位数（如 `045_标题.md`），确保导出的 Markdown 文件在资源管理器中与手机上的时间线保持严格的**一致性排序**。同时截断超长废话标题至 50 字符，杜绝路径溢出。

### 3. Platinum 富文本渲染引擎 (Markdown Compiler)
彻底瓦解小米笔记的 `data_list` 私有富文本标签流，转化为标准的高质量 Markdown 语法：
- **全量排版支持**：精准映射一/二/三级标题、加粗、斜体、下划线 `<u>`、删除线 `~~`、高亮背景 `<mark>` 以及对其方式 `<div align="...">`。
- **多级嵌套**：智能解析单选框 `- [x]` 状态，完美还原无序/有序列表的多级深度缩进。
- **链接转译**：私有 `<a>` 标签无缝转化为标准 `[文字](URL)` 格式。

### 4. HTML5 原生媒体沉浸 (Media Native Integration)
告别原始的 `[音频附件]` 文本占位，引擎引入了高级 Markdown 交互：
- 识别内联标记 `<sound>` 和 `<video>`，自动桥接至本地 `assets/` 目录。
- 注入原生 HTML5 `<audio controls>` 与 `<video controls>` 标签，确保您可以在支持 Markdown 的现代软件（Obsidian, Typora 等）中**直接点击播放语音和视频**，体验媲美原生 APP。
- 配置了专属的正则清理器“白名单”（包含 `u`, `mark`, `div`, `audio` 等），在抹除 MIUI 冗余 XML 标签的同时完美保护生成的渲染样式不受破坏。

## 📂 快速开始指南

### 1. 准备阶段
准备好 MIUI 一键导出的完整备份包（例如：`D:\...\20260427_023118.zip`）。

### 2. 运行环境
- Python 3.x 标准库（**无需安装任何第三方依赖**，零污染运行）。
- OS 跨平台兼容：Windows / macOS / Linux。

### 3. 一键穿甲导出
您只需修改或者通过命令行传递备份文件路径：
```bash
python main.py "您的备份压缩包路径.zip"
```
终端将实时输出进度，完成后会在同目录下生成高保真的 `markdown/` 和 `assets/` 资料库。
